Project features an unified interface for all of its parts. The main idea is to keep all the controls on one screen, while showing only what is necessary.

Tabs

The unified interface features 7 buttons for switching between tabs. The content of these tabs is specific for each application:

The first tab always has the same contents and is used for login and settings.

This tab contains application selector and description of the selected app.

This is the main tab present in all applications, it contains query parameters.

The tools tab is meant for data editing tools, usually only advanced authorized users need it.

This tab is used for plot settings, i.e. to change temperature unit.

This tab is used for data export.

The graph tab is always present, so this button switches layout, making plot bigger.

Login

The authorization is used to restrict users permissions for accesssing some applications or changing data, the login is not required for ordinary use.

Settings

In the info tab you can change your theme and layout, I personally recomend you to use the Dark theme and Rich layout.
